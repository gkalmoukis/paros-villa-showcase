## Overview
A stunning, single-page pre-launch landing page for a luxury villa rental brand in Paros & Antiparos, Greece. Mediterranean luxury aesthetic with elegant animations, custom color palette, and Google Fonts (Inter + Playfair Display).

## Design System
- **Colors**: Deep blue (#1B3A5C), light blue (#6BA3D6), seafoam green (#5CB88A), white, off-white (#F5F7FA)
- **Typography**: Playfair Display for headings, Inter for body text
- **Style**: Clean whitespace, subtle fade-in-on-scroll animations, luxury boutique hotel feel

## Page Sections (top to bottom)

### 1. Hero Section
- Full-viewport background with Unsplash image of a Greek luxury villa
- Dark gradient overlay for readability
- "Welcome to Paros Villa Hub" heading + subtitle
- "Discover More" CTA button with smooth scroll to next section

### 2. Introduction Section
- Elegant paragraph about the brand's curated collection of villas

### 3. Feature Cards (2x2 / 4-column grid)
- Private Villas, Concierge Services, Island Experiences, Private Pools & Sea Views
- Each with a Lucide icon, title, and short description
- Subtle hover effects

### 4. Welcome Message Section
- Off-white background for visual separation
- Personal welcome message from the team
- Sign-off by Konstantinos Tsantilas

### 5. Why Paros Villa Hub Section
- 6 key differentiators in a clean grid layout
- Each with an icon and short description

### 6. Events & Productions Banner
- Full-width section with background image
- Heading about weddings, events & film productions

### 7. Property Owners Section
- Light green-tinted background
- Three feature items: Promotion & Bookings, Property Management, Tailor-Made Services

### 8. Footer
- Brand name + tagline

- Copyright line

## Technical Details
- Mobile-first responsive design
- Intersection Observer fade-in animations on scroll
- Smooth scroll behavior
- SEO meta tags + Open Graph tags
- Unsplash placeholder images for all visual sections
